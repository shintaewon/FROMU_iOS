# 사람냄새나는 커플 교환일기 및 커뮤니티 서비스

> CMC - 12th 프롬유팀 FROMU

연인과의 교환일기, 다른 커플들과의 커뮤니티 서비스를 제공합니다.

사람냄새나는 서비스를 만들고 싶은 5명이 모였습니다.

저흰 타겟을 20-30대 커플로 잡아 커플을 위한 따뜻한 소통 서비스를 기획하였습니다.
